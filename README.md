## Installation

* [Bower](http://bower.io/): `bower install --save blocss-plugin.givebreakpoint`
* Download: [zip](https://github.com/Blocss/plugin.givebreakpoint/zipball/master)
* Git: `git clone https://github.com/Blocss/plugin.givebreakpoint`
